### Представление данных

Данные представлены в следующем формате:
```
1 3 5
2 4 6
3 1 10
2 4 -1
1 2 8 
```

Означает 1 --3--> 5, т.е. ребро из вершины 1 в вершину 3 с весом 5 (src, dst, weight). Также есть возможность использования отрацательных весов. При наличии цикла отрицательного веса будет выведена ошибка, т.к. наличие цикла отрицательного веса лишает задачи смысл.

### Запуск программы
Сначала происходит подсчет количества вершин в `find_vertex_count.py`. После этого значение подставляется в `find_shortest_way.py`. Это значение необходимо, т.к. количество циклов map-reduce = vertex_count - 1.

Запуск осуществляется через bash-script `run.sh`:
```
./run.sh <start_vertex> <data.txt>
```
